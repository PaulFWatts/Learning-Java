Overview

Using individual variables puts limits on the kinds of problems that can be solved with programming. Being able to work with collections of data removes that limitation. Arrays are ordered collections of data of the same type, and allow you to have a single variable name that represents many individual pieces of data.

This series of videos introduces arrays and the key syntax for working with them, shows you how to build on your code tracing skills to trace code that involves arrays, discusses some features of methods for working array data, and finally introduces the advanced topic of multidimensional arrays.

Total length: 40 minutes

Aim to watch before: Week 5 lecture
