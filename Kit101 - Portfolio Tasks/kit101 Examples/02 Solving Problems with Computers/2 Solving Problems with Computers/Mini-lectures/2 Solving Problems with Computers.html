Overview

An introduction to solving problems with computers (using algorithms) and how you can do that in the Java programming language. These videos align with the tasks starting in Week 2, although you'll be using these skills throughout the semester. The videos introduce Algorithms, Using 'Primitive' Data (numbers and individual characters), Importing & Creating Objects (pre-written code that provides useful functionality for your programs), and Some Commonly Used Objects (that you will use in many of your programs to come).

Total length: 97 minutes

Aim to watch before: Week 2 lecture
