Overview

These activities give you practise in reading and writing code makes decisions about what to do.

Objectives

After completing these activities you should be able to:

  1. Plan and implement algorithms that use:
    • 2-way branches
    • multiway branches
  2. Understand the possible ways of achieving branches in programs, be able to choose the most appropriate method to use to solve a particular problem, and write the code to implement this.
    • single if statement (with or without else)
    • nested if statements
    • a sequence of if statements
    • switch statement

Preparation

  1. Review lectures and notes on branching control flow.
  2. Download the starter files (zipped folder) for these activities. Different files are used in different activities, but it is faster to download the full set all at once. Make sure you unzip the files to your work area.

  3. Check that you can write the “template” code for the following. It is STRONGLY recommended that you write code using these constructs by typing the template and then filling in the gaps (with the expressions and statement you have worked out in your planning).

    Construct Template
    if statement (no else) 
    if ( boolean expression ) {
    statement(s)
}
    if statement (with else) 
    if ( boolean expression ) {
    statement(s)
} else {
    statement(s)
}
    switch statement 
    switch ( expression ) {
    case value : statement(s)
                    break;
    default: statement(s)
}
    As of Java 7, the range of suitable variable types for use in a switch statement are byte, short, char and int primitive data types, enumerated types and the String class.

Activities
