Overview

The purpose of this activity is to make sure that you are familiar with some ideas and operations that are essential to all the programming you will do during the semester (and most other programming as well).

These early stages can be very frustrating—the best hint is to do “little and often”. Come into the lab most days and do a little bit until the mechanical tasks of logging on, getting the work space set up, writing and compiling a program become just that: mechanical.

The up side is that once these tasks become mechanical you can use your brain power to think about the interesting aspects of writing programs to solve problems.

Objectives

After completing this activity you should be able to:

  1. Log on to the system and set up the desktop ready to work on programming exercises.
  2. Create, copy, move, and delete files from your work area (where you save your work is up to you, but keeping your files organised will help you find useful examples for revision later).
  3. Recognise the error messages produced by common errors seen in writing, compiling, and running Java programs.
