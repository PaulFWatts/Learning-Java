Expected outcomes

  • Be able to recognise from class documentation the methods that are class methods
  • Be able to write code that correctly uses calls to class methods
  • Be able to recognise from class documentation the data that is stored in a class
  • Be able to write code that correctly uses class data

Starting point

If you haven’t already, download and unzip the Self-study 3 starter files. You will be using MathUsage.java in this activity.

Steps

Use data and methods in the Math class

Open the file MathUsage.java.

Identify statements that do the following:

Action Statement
calculating the difference between myBooks and yourBooks
(where is the answer stored?) 
difference = Math.abs(yourBooks - myBooks);`

This statement actually does two things, in this order:

  1. The difference is calculated: yourBooks - myBooks
  2. The abs method of the Math class is used to calculate the absolute value of this difference (which turns negative values into equivalent positive values)

The “answer” to all of this is stored in the int variable difference.

The square of pizzaDiam/2 is calculated as part of calculating the pizza’s area:

pizzaArea = Math.PI * Math.pow(pizzaDiam/2, 2);

The part that is responsible for calculating the square is the call to Math.pow() with the arguments pizzaDiam/2 and 2. Math.pow(n, p) calculates the value of np.
