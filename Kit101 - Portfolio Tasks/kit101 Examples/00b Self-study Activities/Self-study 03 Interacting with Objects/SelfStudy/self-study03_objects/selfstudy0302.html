Create and use an object of the Poet class

The Poet class is a small program component for reproducing poems by a small number of authors. It is part of the KIT101 Support Library, which also includes the Turtle. Read the online documentation for the Poet class, which can be found at Introductory Programming Notes / KIT101 Support Library API Documentation on MyLO.

In this task you will use the PoetMenu.java file. Open it with DrJava and make sure that you understand what each statement in this program will do (although there isn’t much there yet).

In this first task you’ll create and use a Poet object to make a non-interactive program that displays a menu of options, but always continues as if option 1 has been selected. You can make it interactive later.

So the complete, but non-interactive program is: 
import kit101.Poet;

/**
 * Provides an interface to the Poet class from the KIT101 support library.
 */
public class PoemMenu {

    public static void main(String[] args) {
        Poet bard; //the source of poems
        String menu, poem; //the menu of options and retrieved poem
        
        //Create Poet and display menu of options
        bard = new Poet();
        menu = bard.getTopics();
        System.out.println(menu);
        //Display a particular selection
        poem = bard.getPoem(1);
        System.out.println(poem);
    }

}

Make the Poet program interactive

At the moment the PoetMenu program gives the appearance of being interactive but doesn’t actually allow the user to select which poem to display. Fix this.

  • At the top of the source file, import the Scanner class so you can read the user’s input import java.util.Scanner;
  • Declare and instantiate a Scanner object Scanner sc = new Scanner(System.in);
  • Declare a new int to hold the user’s menu choice int choice;
  • After the menu is displayed, prompt the user to enter their choice of poem For example, System.out.print("Which poem would you like? "); and then,
    choice = sc.nextInt();
  • Replace the hard-coded 1 in the call to getPoem() with the user’s choice poem = bard.getPoem(choice);
Which makes the complete, interactive program: 
import kit101.Poet;
import java.util.Scanner;

/**
 * Provides an interface to the Poet class from the KIT101 support library.
 */
public class PoemMenu {

    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        Poet bard; //the source of poems
        String menu, poem; //the menu of options and retrieved poem
        int choice; //user's choice
        
        //Create Poet and display menu of options
        bard = new Poet();
        menu = bard.getTopics();
        System.out.println(menu);

        //Get user's choice
        System.out.print("Which poem would you like? ");
        choice = sc.nextInt();

        //Display user's selection
        poem = bard.getPoem(choice);
        System.out.println(poem);
    }

}
