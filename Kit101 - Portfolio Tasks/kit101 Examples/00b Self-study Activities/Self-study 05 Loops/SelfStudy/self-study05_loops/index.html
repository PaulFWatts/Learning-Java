Overview

These activities give you practise in reading and writing code that contains loops.

Objectives

After completing these activities you should be able to:

  • Recognise the components of a loop in code
  • Plan, implement, test, and debug programs that use loops to solve problems
  • Plan, implement, test, and debug programs that use nested flow of control to solve problems

Preparation

  1. Review the relevant lectures and notes.
  2. Download the starter files (zipped folder) for these activities. Different files are used in different activities, but it is faster to download the full set all at once. Make sure you unzip the files to your work area.

  3. Check that you can write the “template” code for each of the loop constructs in Java.

    while loop

    while ( boolean expression ) {
    statement(s)
}

    I: before code of loop
    T: in ( )
    B: in block { }
    U: in block

    Note: As this is a pre-tested loop, terms in the test must have a value when the loop starts

    do while loop

    do {
    statement(s)
} while ( boolean expression );

    I: before code of loop
    B: in block { }
    U: in block
    T: in ( )

    Note: As this is a post-tested loop it will always execute at least once

    for loop

    for (initialisation; boolean expression; increment) {
    statement(s)
}

    I: first statement in ( )
    T: second statement in ( )
    B: in block { }
    U: third statement in ()

    Note: This is a pre-tested loop

Activities

  1. Identify components of loops in existing program code
  2. Plan, implement, and test loop algorithms
  3. Plan algorithms using nested control structures
