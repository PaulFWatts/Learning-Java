Plan and implement an algorithm that uses nested flow of control

In this activity you will create and then refine an algorithm that combines iteration and branching in three development steps. When you start implementing the algorithm, do so in a class called NumberEntry (i.e., in a file called NumberEntry.java).

  1. Plan an algorithm that will allow the user to enter integers, and entry of -999 makes the program stop. Echo each integer that the user enters (except the final -999) to the screen using System.out.println(). (Hint: this will need a loop, so think about (i.e., write down) the I B U T.)

    data storage:
    int variable for the entry

    loop:
    T: entry not -999 (best to test this before entering loop - because the -999 is not to be processed)
    I: will need to declare a variable and get first entry from user
    B: echo the entry to the screen
    U: get next entry from user

    Note: It would be a nice idea to use a final variable for the “sentinel” value -999 (the one that stops the loop)
    Implement and test the algorithm. Do not proceed until this works correctly.

    (in main)

    int entry;
final int SENTINEL = -999;
final String PROMPT = "Please enter an integer, " + SENTINEL + " to stop: ";

System.out.print(PROMPT);
entry = sc.nextInt();
while (entry != SENTINEL) {
    System.out.println("number entered: " + entry);
    System.out.print(PROMPT);
    entry = sc.nextInt();
}
  2. Modify the algorithm so that the user is given a message about each of the integers they enter (except the final -999). The message should say whether the integer is odd or even.

    This is something that will need to be added to the body. To test whether the entry is odd or even remember that even numbers are divisible by 2 (i.e., the remainder after dividing by 2 will be zero).

    Implement and test this. Do not proceed until it works correctly.

    (in main)

    int entry;
final int SENTINEL = -999;
final String PROMPT = "Please enter an integer, " + SENTINEL + " to stop: ";

System.out.print(PROMPT);
entry = sc.nextInt();
while (entry != SENTINEL) {
    System.out.println("number entered: " + entry);
    if (entry % 2 == 0) {
        System.out.println("entry is an even number");
    } else {
        System.out.println("entry is an odd number");
    }
    System.out.print(PROMPT);
    entry = sc.nextInt();
}
  3. Modify the algorithm so that when the user finishes entering integers they are shown a message indicating how many odd numbers they entered and how many even numbers.

    This will need extra data storage: ints to hold the number of odd and of even numbers. These will need to be initialised to 0 and incremented in the loop (in the if statement).

    Display of the values will happen after the loop is over.

    Implement and test this.

    (in main)

    int entry;
int odds = 0;
int evens = 0;
final int SENTINEL = -999;
final String PROMPT = "Please enter an integer, " + SENTINEL + " to stop: ";

System.out.print(PROMPT);
entry = sc.nextInt();
while (entry != SENTINEL) {
    System.out.println("number entered: " + entry);
    if (entry % 2 == 0) {
        System.out.println("entry is an even number");
        evens = evens + 1;
    } else {
        System.out.println("entry is an odd number");
        odds = odds + 1;
    }
    System.out.print(PROMPT);
    entry = sc.nextInt();
}
System.out.println(odds + " odd numbers");
System.out.println(evens + " even numbers");
