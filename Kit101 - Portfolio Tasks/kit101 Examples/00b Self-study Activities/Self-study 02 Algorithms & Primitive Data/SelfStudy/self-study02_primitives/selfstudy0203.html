Expected outcomes

You will be able to use the Scanner class to read values entered from the keyboard and store them in variables of primitive types.

Background information

All programs that use the Scanner class must include an import statement:

import java.util.Scanner;

Note that it is also possible to instruct the compiler to import everything from the java.util package, with import java.util.*, but this is not good programming practice because (1) it makes compilation slower and (2) as you use more and more packages it becomes more likely that two imported classes (only one of which you actually want to use) will have the same name.

To use methods in the Scanner class it is necessary to create a Scanner object. This should usually be done inside the main() method.

Some of the useful methods that Scanner objects have are described in lecture 2, and below. You could even read the Scanner class’s API documentation online but it is highly detailed and not necessary at this stage.

Adding “interactivity”

To allow a user to enter a value into a variable, we need to make sure that our code does four things:

  1. creates a Scanner object (note that a single Scanner object can be used to input many values into variables of different types).
    Then, for each value to be input via the keyboard:
  2. “prompts” the user (that is, tells them what to enter)
  3. accepts the input
  4. stores the value that the user enters into the correct variable.

Prompting

This is done using a statement of the form

System.out.println("This is the prompt");
or
System.out.print("Input will appear on the same line as this prompt: ");

before calling the method in the Scanner object.

Accepting the input

This is what methods belonging to a Scanner object do for us. Remember that you must use a different method for each variable type:

nextInt() to accept an int value
nextDouble() to accept a double value
nextBoolean() to accept a boolean value

Storing

This is done with an assignment statement. Remember the assignment operator is = (call this “becomes”)

Putting it together

Example 1: Suppose we have a program with a boolean variable called over declared, and that scan refers to a Scanner object. We could prompt the user to enter a boolean value and store it in the variable like this:

System.out.println("is it over? true or false");
over = scan.nextBoolean();

Example 2: Suppose we have a program with two int variables called num1 and num2 declared, and that scan refers to a Scanner object, we could use a single prompt to have the user enter values into both of these variables like this:

System.out.println("Please enter two integers");<br>
num1 = scan.nextInt();<br>
num2 = scan.nextInt();<br>
