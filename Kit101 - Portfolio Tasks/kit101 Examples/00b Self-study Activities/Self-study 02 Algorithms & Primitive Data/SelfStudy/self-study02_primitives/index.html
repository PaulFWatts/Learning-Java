Overview

The purpose of these activities is to practise working on algorithms (the steps you want the program to do) and then implementing those algorithms as program code. As you get better at reading and writing low-level program statements (the second step) you will also get better at thinking about and writing algorithms, because you’ll know how the solution to a problem needs to be broken down so that a computer can process it.

Objectives

After completing these activities you should be able to:

  1. Work out high level and low level algorithms to solve simple problems.
  2. Write Java code to solve a problem by declaring and manipulating variables of primitive types. (This is implementing a low level algorithm.)
  3. Write Java source code to allow a user to enter values into variables of primitive types.

Download the starter files (zipped folder) for this module. Different files are used in different activities, but it is faster to download the full set all at once. Make sure you unzip the files to your work area (which should not be the Downloads folder).

Activities

  1. Plan algorithms
  2. Read, understand, and write Java code to solve a problem using data of primitive types
