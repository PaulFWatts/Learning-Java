Overview

Methods are a way of 'wrapping up' and giving a name to a collection of statements that achieve some useful function so that those actions can be reused without needing to physically repeat the statements in our code. The first video in this collection introduces the syntax for defining a method to be called directly from main(), while the second discusses what happens when a method is called (from main or another method).

Total length: 35 minutes

Aim to watch before: Week 3 tutorials and lecture
